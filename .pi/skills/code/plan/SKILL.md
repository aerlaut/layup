# Plan
Use this skill when the user asks to make plan.

## Steps
1. Understand the plan the user wants to make, and ask for more details if necessary.
2. Understand the codebase and the problem to be solved. Ask for more details if necessary.
3. Suggest the best plan to carry out the requestd changes. Break down the plan into smaller isolated independent tasks that can be done in parallel. Make sure to include all necessary steps to complete the plan.
